## Введение
Проект для работы с BMS платами от производителя Jikong через Bluetooth 
интерфейс. 
Разрабатываемый (базовый) функционал - это сбор данных с платы для дальнейшей 
работы с ними.

Предполагаемый функционал для реализации:
* сбор данных по ячейкам аккумулятора (напряжения, емкость токи и пр.) в базу 
данных
* выгрузка (или прямой вывод) данных в Excel-таблицу
* создание собственного API для возможности программно транслировать данные 
через сеть, другие процессы ПК и т.д.
* изучение зависимости физических величин и состояния ячеек аккумулятора
* попытка разработать модель для предсказания состояния аккумулятора

## Как пользоваться?
### Установка
Перед использованием необходимо настроить программную среду. 
Для этого надо запустить [скрипт](scripts/setup_windows.bat), если 
вы работаете в ОС Windows 11 или [скрипт](scripts/setup_linux.sh) если в 
Linux.

Скрипт сам установит `python` и создаст виртуальную среду.

### Запуск
Перед тем как запустить программу необходимо:
* ввести плату BMS в рабочий режим, включить на ней Bluetooth
* на ПК также включить Bluetooth

Далее через терминал Windows запустить 
[скрипт](scripts/.venv/Scripts/activate.bat) для активации 
виртуальной среды для `python`.

Потом запустить сам `python` скрипт [main.py](src/main.py)


## Техническая информация
### Инструменты программирования
Проект написан на Python с использованием библиотеки `bleak` и предназначен 
для запуска на ПК с Windows 11 или Linux (Ubuntu/Debian).

При этом ПК должен иметь Bluetooth интерфейс.
